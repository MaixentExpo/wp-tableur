== Description ==

Framework de gestion de tables dans Wordpress.

Le descriptif des tables est à définir dans un ficier de config et le plugin fait le reste.
Il va proposer 
- la table dans une liste colonnée (tableur)
- des actions pour entre autres créer/modifier/supprimer les éléments

Le  plugin utilise la classe WP_List_Table de Wordpress.

== Installation ==

1. Recopier le répertoire du plugin dans '/wp-content/plugins/' 
2. Activer le plugin dans les extensions de l'administration de Wordpress
3. Le plugin va créer les tables et le menu d'administration des tables
4. Cliquer sur admin / Tableur
